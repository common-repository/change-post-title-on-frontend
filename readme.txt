"Change Post Title on Frontend" allows you easily to change post titles without entering the "Edit post" section.

== Description ==
This plugin allows you to easily change post titles on the frontend without having to enter the "Edit post" section.

**Note:** In order for this plugin to work, your titles **MUST BE** wrapped in an H1 tag! Also, this plugin doesn't work on links, only on pure text!

How to use:

1. Double-click on the post title
2. Enter the new title
3. Hit enter

If you have any suggestions or feedback, get in touch with me via stefan.malic@gmail.com.

== Installation ==
1. Upload the plugin .zip file to the "/wp-content/plugins/" directory OR install it directly through the Wordpress' Plugins section.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. All done!

== Frequently Asked Questions ==

= Will this work on any website? =

It will only work if the post titles on your website are wrapped in an H1 tag.

= Can I define my own title wrapper? =

At this stage no. It is planned for the next version though.

= Will this plugin work on all posts? =

Only on those that aren't wrapped in an <a> tag.
